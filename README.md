# Projeto iCantina

## Descrição do Projeto

O Projeto ICantina foi desenvolvido no âmbito do projeto da disciplina de Desenvolvimento de Aplicações. Pretendia-se que fosse feito um programa para gerir menus e reservas de uma cantina escolar, carregar o saldo dos clientes e efetuar a faturação das reservas.
